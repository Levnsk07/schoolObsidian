# Замечательный предел

## Второй Замечательный предел
$$
\lim_{x \rightarrow \infty}(1 + \frac{1}{x})=1 
$$
# Свойства из *второго замечательного предела*

1. $\lim_{x \rightarrow 0}(\frac{\ln(1+x)}{x}) = 1$
2. $\lim_{x \rightarrow 0}(\frac{a^x - 1}{x}) =\ln a$
3. $\lim_{x \rightarrow 0}(\frac{(1+x)^a - 1}{x}) = a$
4. $\lim_{x \rightarrow 0}(1 + \frac{a}{x})^x = e^a$
