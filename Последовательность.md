# Определение
- **Последовательность** - [[Функция]] $a_n : N \rightarrow R$ из множества натуральных чисел в множество действительных чисел называется *последовательностью*. А значениями $a_n:N \rightarrow R$  называют элементами последовательности.
# Операции
- **Произведение последовательности на число** - каждый элемент последовательности умножается на некоторое $k$ 
- **Суммой последовательности** - $A$ и $B$, которые одинаковой длинны, называется новая последовательность $C$, где элемент определяется как: $c_n = a_n +b_n | a \in A, b \in B$
