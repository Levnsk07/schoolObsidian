- В [[Ориентированный Граф|ориентированном графе]], **полустепень захода** вершины $v$, называют [[Степень вершины|степень]] только для [[Ребро|рёбер]] для которых точка $v$ является начальной.
> То есть количество исходящих рёбер 
- Обозначается как: $\deg^-(v)$
