# Теорема
- Пусть дана последовательность [[Вложенные Отрезки|вложенных отрезков]] $[a_n,b_n]$, тогда, если длинны отрезков стремятся к нулю $\lim_{n \rightarrow \infty} (b_n - a_n) = 0$, то существует единственная точка $c$, принадлежащая всем отрезкам:
$$
c \in \bigcap_{n=1}^{\infty} [a_n, b_n]
$$
# Доказательство 
1. [[Монотонные Последовательности|Монотонность]]
	- Последовательность левых концов $\{a_n\}$ монотонно [[Неубывающая Последовательность|не убывает]]
	- Последовательность правых концов $\{b_n\}$ монотонно [[Невозрастающая Последовательность|не возрастает]]
2. Ограниченность 
	- $\{a_n\}  [[Ограниченная Последовательность|ограниченна]] любым $b_n$
	- $\{b_n\}  [[Ограниченная Последовательность|ограниченна]] любым $a_n$
3. Равенство пределов 
	- Поскольку $a_n < b_n$ для всех $n$, то и $c \le d$. Но по условию $\lim_{n \rightarrow \infty} (b_n - a_n) = 0$, а значит $d-c = 0 \rightarrow d =c$ 
4. Предположим существует другая общая точка $d \neq c$, тогда $|d - c| > 0$, **НО** так как длинны стремятся к нулю, то существует такое $N$, что для всех $n \ge N$ длинна $[a_n,b_n]$ меньше $|d-c|$ то есть $d$ не может принадлежать всем вложенным отрезкам $\rightarrow$ **Противоречие**
- **Вывод**: Может существовать только одна подобная точка.
