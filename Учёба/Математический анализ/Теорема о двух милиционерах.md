# Теорема
- Если в некоторой проколотой окрестности точки $a$ выполняется не равенство с [[Функция|функциями]] $h(x) \le f(x) \le g(x)$ и их [[Предел|пределы]] $\lim_{x \rightarrow a}{h(x)} = \lim_{x \rightarrow a}{g(x)} =b$, то $\lim_{x \rightarrow a}{f(x)} = b$
# Доказательство 
## Дано:
- Даны три [[Функция|функции]], которые: $h(x) \le f(x) \le g(x)$,  где $h(x)$ и $g(x)$ -- "*милиционеры*"
## Ход решения
1. По условию оба "*милиционера*" стремятся  к общему пределу: $\lim_{x \rightarrow a}{h(x)} = \lim_{x \rightarrow a}{g(x)} =b$ 
2. По определению [[Предел Функции#Предел по Коши|Предела]], для любого $\epsilon >0$ существуют такие окрестности точки $a$, в которых выполняются условия: 
	1. $b- \epsilon < h(x) < b +\epsilon$
	2. $b- \epsilon < g(x) < b +\epsilon$
3. Так как  $h(x) \le f(x) \le g(x)$, то в этой же окрестности выполняется: $L - \epsilon < h(x) \le f(x) \le g(x)< L + \epsilon$ 
4. **ВЫВОД**: По определению [[Предел Функции#Предел по Коши|Предела]] из неравенства: $L-\epsilon<f(x)< L+\epsilon$, следует $|f(x) -  L|<\epsilon$
