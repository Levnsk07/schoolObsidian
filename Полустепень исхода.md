- В [[Ориентированный Граф|ориентированном графе]], **полустепень исхода** вершины $v$, называют [[Степень вершины|степень]] только для [[Ребро|рёбер]] для которых точка $v$ является конечной.
> То есть количество входящих рёбер 
- Обозначается как: $\deg^+(v)$
