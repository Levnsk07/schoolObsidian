# Теорема
- Если [[Последовательность]] $\{a_n\}$ [[Монотонные Последовательности |монотонна]] и [[Ограниченная Последовательность |ограниченна]], то она имеет конечный [[Предел]].
# Доказательство 
- **Дано**: [[Последовательность ]] $\{a_n\}$, [[Неубывающая Последовательность|неубывающая]] и [[Ограниченная Последовательность|ограниченная сверху]] 
1. Пусть $S = \sup{(a_n)}$ , то есть $S$ - [[Точная граница|точная]] верхняя граница
2. Поскольку [[Последовательность]] неубывающая, для всех $n \ge N$ выполняется:
	$$	a_n \ge a_N > S-\epsilon	$$
	- С другой стороны $a - n \le S | \forall_n$ таким образом для всех $n \ge N$:
	$$S- \epsilon < a_n \le S < S + \epsilon$$
3.  То есть $|a - n| \le \epsilon$ | $\forall _n \ge N$
- **Доказано**
